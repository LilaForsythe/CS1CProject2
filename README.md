# Project-2-
Design and implement a Digital Library Management System in C++ using object-oriented programming principles. Your program should model a library that contains different types of media and allows users to manage, access, and interact with them.

# Project's Requirements: 
  Your solution must include:
   - multiple classes
   - inheritance
   - at least one abstract base class
   - at least one interface using pure virtual functions
   - virtual functions and runtime polymorphism
   - exceptions for invalid operations
   - proper encapsulation
   - separate header (.h) and implementation (.cpp) files
